# 基于QT的多媒体播放器

## 需求分析

1、支持视频播放、音频播放、图片预览、文本编辑，支持市面上大多数主流文件格式。
2、支持播放列表。
3、音、视频播放支持倍速，快进/退，支持拖动进度条。
4、播放器支持常用的快捷键。
5、图片浏览器支持捕获剪切板复制的图片内容，并支持以多种文件格式进行保存，支持将打开的图片以图片形式复制到剪切板。
6、文本编辑器支持多种格式文件的编辑与保存，包括纯文本与富文本。
